# Project-13
Dự án Full-stack Blog Application phát triển bằng MERN stack (MongoDB, Express, React, Node.js). Hệ thống tích hợp RESTful API chuẩn, xác thực bảo mật JWT. Frontend hiện đại, responsive với Tailwind CSS; hỗ trợ đầy đủ tính năng quản lý bài viết (CRUD), bình luận, thả tim, phân trang và bộ lọc tìm kiếm tối ưu UI/UX
